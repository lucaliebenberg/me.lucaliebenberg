Personal projects

I'm currently contributing (or have contributed) my time to the following projects:

  • $basedLAND

    Introducing a web 3.0 focussed project from a environmental awareness hackathon. This platform allowed users to participate in the token sale, stake their tokens, and receive rewards, thereby getting value from their investments.

  • Hotel Reservation

    Hotel reservation JSON api that allows user to authenticate themselves, list hotels, book available hotels, list rooms, cancel bookings, seed scripting - implementing admin authentication on specific routes.

  • Traffic Toll Microservice

    A microservice built for calculating invoices for truck delivery companies, based on distance travelled during delivery windows.

  • Post It

    A fullstack web application for creating and managing posts, aimed at helping those who hand out job slips at traffic robots in South Africa by providing a self-service platform to share their slips.
